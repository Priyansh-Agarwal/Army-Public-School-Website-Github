## Run Locally

Clone the project

```bash
  git clone https://github.com/Priyansh-Agarwal/apsambala.github.io.git
```

Go to the Main Folder

```bash
  cd Army Public School Website (NEW)

```

Start the server

```bash
  Live Server (VS Code)
```

  
## Tech Stack

**Client:** HTML, CSS, Bootstrap, and JavaScript.

**Server:** PHP, Asp.Net
